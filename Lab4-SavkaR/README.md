# 📘 Лабораторні роботи з PSoC

## 📌 Загальний опис

Цей репозиторій містить лабораторні роботи з дисципліни **«Програмування систем на кристалі»**.

У рамках лабораторних робіт розглядаються:

- робота з мікроконтролерами PSoC
- використання GPIO
- робота з матричною клавіатурою
- використання зсувних регістрів (74HC595)
- керування семисегментними індикаторами
- використання UART для відлагодження

3. Вибрати файл `.cywrk`

   ```
   Debug → Program (Ctrl + F5)
   ```

Використання зсувних регістрів та семисегментних світлодіодних індикаторів у середовищі PSoC Creator

## 🎯 Мета

- Вивчити роботу семисегментних індикаторів
- Реалізувати вивід даних через 74HC595
- Отримати практичні навички роботи в PSoC Creator

---

## 🧠 Теоретичні відомості

### 🔹 Зсувний регістр

Зсувний регістр — це пристрій, який дозволяє послідовно вводити дані та виводити їх паралельно.

Основні сигнали:

- `CLK` — зсув даних
- `LATCH` — фіксація даних на виході

---

### 🔹 Семисегментний індикатор

Складається з 7 сегментів (A–G) та використовується для відображення цифр і символів.

---

## 🛠️ Виконання роботи

У ході роботи було реалізовано:

- вивід цифр на семисегментний індикатор
- зчитування натискань з матричної клавіатури
- відображення натиснутої кнопки
- підтримка букв: **A, b, C, d, E, F**
- перемикання режимів (цифри / букви)

---

## 💻 Фрагменти коду

### Масив символів

```c
static uint8_t LED_NUM[] = {

    0xA1, //d
    0x86, //E
    0x8E  //F
};
```

---

### Відправка даних у зсувний регістр

```c
static void FourDigit74HC595_sendData(uint8_t data) {
    for(uint8_t i = 0; i < 8; i++) {
        if(data & (0x80 >> i))
            Pin_DO_Write(1);
        else
            Pin_DO_Write(0);

        Pin_CLK_Write(1);
        Pin_CLK_Write(0);
    }
}
```

---

### Відображення символу

```c
FourDigit74HC595_sendOneDigit(0, digit, dot);
```

## ▶️ Інструкція запуску

1. Відкрити проєкт у PSoC Creator
2. Перевірити налаштування пінів:
   - `Pin_DO`
   - `Pin_CLK`
   - `Pin_Latch`
3. Зібрати проєкт (Shift + F6)
4. Завантажити на плату (Ctrl + F5)
5. Натискати кнопки на клавіатурі
