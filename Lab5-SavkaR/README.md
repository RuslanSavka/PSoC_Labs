## 📌 Загальний опис

Цей репозиторій містить лабораторні роботи з дисципліни **«Програмування систем на кристалі»**.

У рамках лабораторних досліджуються:

- робота з мікроконтролерами PSoC
- керування семисегментними індикаторами
- використання зсувних регістрів
- динамічна індикація
- обробка клавіатури
- робота з таймерами та перериваннями
- аналіз сигналів за допомогою логічного аналізатора

# 🧪 Лабораторна робота №5

## 📌 Тема

Динамічна індикація, таймери, робота з логічним аналізатором у середовищі PSoC Creator

---

## 🎯 Мета роботи

- Ознайомитися з динамічною індикацією
- Навчитися працювати з таймерами та перериваннями
- Реалізувати керування дисплеєм через зсувні регістри
- Отримати навички роботи з логічним аналізатором

---

## 🛠 Виконані завдання

✅ Виведення числа на дисплей  
✅ Реалізація динамічної індикації  
✅ Ввід паролю через матричну клавіатуру  
✅ Перевірка паролю з індикацією результату  
✅ Реалізація таймера (5 хвилин)  
✅ Циклічний зсув символів  
✅ Робота з перериваннями

---

## 🧩 Основні компоненти

- Семисегментний індикатор
- Зсувний регістр (74HC595)
- Матрична клавіатура
- Таймер (Timer)
- Переривання (ISR)

---

## 💻 Фрагмент коду

```c
CY_ISR(Timer_ISR)
{
    sendOneDigit(current_digit, display[current_digit]);

    current_digit++;
    if(current_digit >= 8) current_digit = 0;

    ms_counter++;
    if(ms_counter >= 1000)
    {
        ms_counter = 0;
        seconds++;
    }
}
```

## ▶️ Інструкція запуску

1. Відкрити проєкт у **PSoC Creator**
2. Зібрати проєкт (**Build**)
3. Підключити плату
4. Завантажити програму (**Program Device**)

---

## 📊 Опис роботи програми

- Дисплей працює у режимі **динамічної індикації**
- Дані передаються через **зсувний регістр**
- Переривання таймера відповідає за:
  - оновлення дисплею
  - підрахунок часу
- Клавіатура дозволяє вводити пароль

### 🔐 Результат перевірки паролю:

- ✅ **Правильний пароль** → мигання індикаторів
- ❌ **Неправильний пароль** → інший патерн мигання
