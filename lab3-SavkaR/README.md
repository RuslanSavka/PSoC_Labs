# 📖 Теоретичні відомості

**Матрична клавіатура** — це пристрій введення, що складається з рядків і стовпців. Кожна кнопка замикає певну пару "рядок–стовпець".

### 🔹 Принцип роботи:

- по черзі активуються стовпці
- зчитуються рядки
- визначається натиснута кнопка

### 🔹 Переваги:

- економія пінів мікроконтролера
- простота реалізації
- можливість масштабування

## 5. Обробка натискань (switch-case)

```c
void handleKey(uint8 key)
{
    LED_Off();

    switch(key)
    {
        case 1:
        case 7:
            LED_R_Write(0);
            break;

        case 2:
        case 8:
            LED_G_Write(0);
            break;

        case 3:
        case 9:
            LED_B_Write(0);
            break;

        case 4:
        case 10:
            LED_R_Write(0);
            LED_G_Write(0);
            break;

        case 5:
        case 0:
            LED_R_Write(0);
            LED_B_Write(0);
            break;

        case 6:
        case 11:
            LED_G_Write(0);
            LED_B_Write(0);
            break;
    }
}
```

## 6. Реалізація пароля

### 🔹 Заданий пароль:

```c
uint8 password[3] = {1, 2, 3};
```

### 🔹 Перевірка:

```c
void checkPassword()
{
    if(inputIndex < 3) return;

    if(input[0] == password[0] &&
       input[1] == password[1] &&
       input[2] == password[2])
    {
        SW_Tx_UART_PutString("Access allowed\r\n");
    }
    else
    {
        SW_Tx_UART_PutString("Access denied\r\n");
    }

    inputIndex = 0;
}
```
