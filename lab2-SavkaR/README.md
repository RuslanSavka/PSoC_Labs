# Теоретичні відомості

**UART (Universal Asynchronous Receiver-Transmitter)** — це асинхронний послідовний інтерфейс передачі даних, який використовується для обміну інформацією між мікроконтролером і комп’ютером.

Основні особливості UART:

- передача даних відбувається послідовно
- не використовується спільний тактовий сигнал
- використовуються дві основні лінії:
  - **TX** — передача даних
  - **RX** — прийом даних

У цій лабораторній роботі використовується **Software Transmit UART** для передачі текстових повідомлень у термінал комп’ютера.

# Виконання роботи

## 1. Створення проєкту

У середовищі **PSoC Creator** було створено новий проєкт та додано необхідні компоненти.

Використані компоненти:

- `SW_Tx_UART` – програмний UART передавач
- `Button` – кнопка для введення сигналу
- `LED_R`, `LED_G`, `LED_B` – світлодіоди для індикації

## 3. Програмна реалізація

Основний код програми:

```c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable; /* Enable global interrupts. */

    SW_Tx_UART_Start();

    SW_Tx_UART_PutCRLF();
    SW_Tx_UART_PutString("Software Transmit UART");
    SW_Tx_UART_PutCRLF();

    uint8_t last_state = 0;

    for(;;)
    {
        // button == 0 -> Button pressed
        // button == 1 -> Button released

        if(Button_Read() == 1 && last_state == 0)
        {
            last_state = 1;

            SW_Tx_UART_PutString("Button released");
            SW_Tx_UART_PutCRLF();

            LED_R_Write(0); // Turn ON
            LED_G_Write(0);
            LED_B_Write(0);
        }
        else if(Button_Read() == 0 && last_state == 1)
        {
            last_state = 0;

            SW_Tx_UART_PutString("Button pressed");
            SW_Tx_UART_PutCRLF();

            LED_R_Write(1); // Turn OFF
            LED_G_Write(1);
            LED_B_Write(1);
        }
    }
}
```

# Інструкція запуску

## Необхідне програмне забезпечення

Для запуску проєкту необхідно:

- **PSoC Creator**
- плата **PSoC**
- USB підключення до комп’ютера
- термінальна програма (наприклад **PuTTY** або **Tera Term**)

---

## Кроки запуску

1. Відкрити проєкт у **PSoC Creator**
2. Скомпілювати програму (**Build**)
3. Завантажити її на плату (**Program**)
4. Підключитися до UART через термінал
5. Натискати кнопку на платі

У ході виконання лабораторної роботи було вивчено принцип роботи інтерфейсу **UART** та його використання для передачі даних між мікроконтролером і комп’ютером у середовищі **PSoC Creator**.
